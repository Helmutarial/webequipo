# Equipo de fútbol

Plataforma web pública y panel privado para la gestión de un equipo de fútbol.

## Estado

Proyecto en fase de definición y preparación técnica.

Consulta el [plan del proyecto](PLAN_PROYECTO.md) para conocer el alcance, las fases y las decisiones de arquitectura.

## Stack previsto

- Next.js y TypeScript
- PostgreSQL y Prisma
- Auth.js
- TailwindCSS y shadcn/ui
- Docker y Docker Compose

## Flujo de trabajo

1. Crear una rama `feature/nombre` o `fix/nombre`.
2. Realizar cambios pequeños y comprobables.
3. Abrir un Pull Request hacia `main`.
4. Fusionar únicamente cuando las validaciones pasen.
